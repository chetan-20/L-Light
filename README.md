# L-Light

2D Platformer type mini game that i made for a game jam where the theme was
Avoid The Light

Use A,D to Move 

Space to Jump

Shift to Slide

MB1 to attack 

Attack The light to disable it.

Used Singleton For Sound.

Used Unty Animator for animation and Transition.

Used URP for Light.
